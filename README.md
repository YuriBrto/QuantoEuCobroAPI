# Quanto Eu Cobro? 🚀

Uma plataforma SaaS moderna para criação e gestão de propostas comerciais profissionais, com foco em freelancers, agências e profissionais autônomos.

---

## 💡 Sobre o Produto

O **Quanto Eu Cobro?** é uma solução que permite criar propostas comerciais elegantes, personalizadas e rastreáveis em poucos minutos.

Com ele, o usuário pode:

- Criar propostas profissionais em minutos
- Escolher entre templates modernos
- Adicionar campos personalizados dinamicamente
- Compartilhar propostas por link público
- Acompanhar visualizações em tempo real
- Ter controle total de histórico e métricas

---

## ✨ Diferencial da Plataforma

A plataforma foi desenhada para ser **mais do que um gerador de propostas**:

- 🎨 Sistema de templates visuais dinâmicos
- 📊 Tracking de visualizações de propostas
- ⚡ Experiência rápida e fluida
- 📱 Design responsivo mobile-first
- 🧠 Interface focada em produtividade
- 🪄 Experiência premium estilo SaaS moderno

---

## 🧱 Arquitetura

O backend foi construído com foco em escalabilidade e boas práticas:

- ASP.NET Core
- Entity Framework Core
- Arquitetura em camadas
- Separação clara de responsabilidades
- Serviços desacoplados
- DTOs para comunicação segura
- JWT Authentication

---

## 🔐 Autenticação

O sistema utiliza autenticação baseada em **JWT (JSON Web Token)** para garantir segurança e controle de acesso.

---

## 📄 Funcionalidades Principais

### Propostas
- Criar proposta
- Editar proposta
- Listar propostas
- Excluir (soft delete)
- Visualização pública por link

### Templates
- Listagem de templates disponíveis
- Associação de template à proposta
- Base visual para renderização futura no frontend

### Dashboard
- Total de propostas
- Propostas visualizadas
- Faturamento estimado
- Resumo de atividade do usuário
